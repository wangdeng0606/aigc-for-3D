{objectsData}

## 坐标系统：
- 右手坐标系（right_hand）
- 单位：meter（米）
- 原点：scene_center（锚点物品的中心）
- X轴：正值向右，负值向左
- Y轴：正值向上，负值向下
- Z轴：正值向内（远离相机），负值向外（靠近相机）

```json
          "position": [0, 0, 0],
          "rotation": [0, 0, 0],
          "scale": [1, 1, 1]
        },
        "size": [1, 1, 1]
      }
    ]
  }
}
```

## 生成规则：
1. 锚点物品的position必须是[0, y, 0]，作为坐标原点
2. 其他物品的坐标相对于锚点物品计算
3. transform.position: [x, y, z] 单位为米
4. transform.rotation: [rx, ry, rz] 单位为度（0-360）
5. transform.scale: [sx, sy, sz] 默认为[1, 1, 1]
6. size: [width, height, depth] 物体包围盒尺寸，单位为米
7. 根据图片中物品的实际空间关系和相对位置生成坐标
8. 必须包含所有选中的物品
9. 坐标值保留2位小数
